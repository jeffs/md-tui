## Implementation Approach

### Files to Modify

1. **Cargo.toml**
   - Add `tree-sitter-diff = { version = "0.1.0", optional = true }` to dependencies
   - Add `"tree-sitter-diff"` to `tree-sitter` feature group

2. **src/highlight/mod.rs**
   - Add `#[cfg(feature = "tree-sitter-diff")] mod diff;`
   - Add match arm: `"diff" => HighlightInfo::Highlighted(diff::highlight_diff(lines).unwrap())`

### Files to Create

1. **src/highlight/diff.rs** (~30 lines)
   - Follow pattern from `rust.rs`
   - Use `tree_sitter_diff::LANGUAGE` and `tree_sitter_diff::HIGHLIGHTS_QUERY`
